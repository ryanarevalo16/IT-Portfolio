# System Inventory Tool (Python)

## Description
This project is a Python-based IT asset inventory tool that collects system information 
and exports it into a structured CSV report for IT tracking and auditing.

## Features
- Collects hostname, OS, CPU, RAM, disk usage
- Captures logged-in user and IP address
- Generates timestamped reports
- Exports data to CSV format

## How to Run
```bash
python3 inventory.py
